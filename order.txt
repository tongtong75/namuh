# 사용자가 로그인을 하면 사용자 회사 일련번호를 이용하여 CO_MNG 테이블에서 회사 정보중 
  `BGNG_YMD` date DEFAULT NULL COMMENT '검진시작일자', `END_YMD` date DEFAULT NULL COMMENT '검진종료일자' 를 
  이용하여 예약신청기간 및 검진가능기간 표시
# 로그인한 사용자의 가 검진예약이 되어 있는지 CKUP_TRGT 테이블에서 검진대상자(BUSINESS_NUM 가 로그인한 사용자와 동일한 경우) 정보를 가져오고 
화면에 검진예약이 되어 있는 경우 예약취소 버튼이 표시되고 검진예약이 되어 있지 않은 경우 예약하기 버튼이 표시됨
